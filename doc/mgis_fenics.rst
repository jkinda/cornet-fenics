Documented demos coupling FEniCS with MFront
============================================

These demos are based on the :code:`mgis.fenics` module of the `MFrontGenericInterfaceSupport (MGIS) project <https://thelfer.github.io/mgis/web/index.html>`_. A general introduction of the package
is `available here <https://thelfer.github.io/mgis/web/mgis_fenics.html>`_. This project has been realized in collaboration with Thomas Helfer (CEA, thomas.helfer@cea.fr).
